## フォーム名
**ByakuyaAI 動画生成依頼フォーム**

## フォーム説明文

> マイソクと物件写真をアップロードいただくと、AI が自動でショート動画を生成し、ご登録の SNS アカウントへ配信します。
>
> 【入力前にご確認ください】
> - マイソク（PDF または JPG/PNG）を1枚
> - 物件写真（ステージング希望分＋そのまま使用分）
> - ステージング希望分：5〜10枚（空室写真）
> - そのまま使用分：5〜10枚（外観・周辺・室内でステージング不要なもの）
>
> 送信後、15〜30分で完成動画がメールにて届きます。

## 技術連携

- **送信トリガー**: Google Apps Script の `onFormSubmit` で n8n Webhook に POST
- **Webhook URL**: `https://aiboost-takeshi.app.n8n.cloud/webhook/fudosan-video`
- **ファイルは**: Drive に自動保存 → ファイルIDを Webhook payload に含める
- **処理完了後**: n8n から完成動画URLをメール通知

## バリデーション（Apps Script側）

- 質問12: ファイル数 5〜10 以外なら送信者にエラーメール＋処理停止
- 質問13: ファイル数 5〜10 以外なら同上
- 質問18: 「いいえ」の場合は処理停止＋送信者に確認メール

- ログイン要求: **OFF**（Googleアカウント不要、担当者変更に対応）
- 進捗バー表示: **ON**
- メール通知: **ON**（管理者 info@byakuyaai.com 宛）

## 作成時の注意

1. 「契約プラン」の条件分岐でセクション4を制御（フォーム作成時に設定）
2. ファイルアップロード欄は各質問ごとに「最大10ファイル」「合計1GB」で設定
3. Apps Script は **フォーム作成後** に「拡張機能 → Apps Script」で以下を追加:

```javascript
function onFormSubmit(e) {
  const response = e.values;
  const formData = {
    company: response[1],
    contact: response[2],
    plan: response[3],
    // ... 他の項目
  };
  UrlFetchApp.fetch(
    'https://aiboost-takeshi.app.n8n.cloud/webhook/fudosan-video',
    {
      method: 'post',
      contentType: 'application/json',
      payload: JSON.stringify(formData)
    }
  );
}
```

その後「トリガー → フォーム送信時」に紐付け。
